python原型链污染

python原型链污染与nodejs原型链污染大致相同，都是需要用merge函数来修改父类的函数，只是nodejs是修改object中的原型中的属性，而python是修改父类中的的属性：

class father:
    secret = "hello"
class son_a(father):
    pass
class son_b(father):
    pass
def merge(src, dst):
    for k, v in src.items():
        print(f"k:{k}\t v:{v}")
        if hasattr(dst, '__getitem__'):
            if dst.get(k) and type(v) == dict:
                merge(v, dst.get(k))
            else:
                dst[k] = v
        elif hasattr(dst, k) and type(v) == dict:
            merge(v, getattr(dst, k))
        else:
            setattr(dst, k, v)
instance = son_b()
print(instance)
payload = {
    "__class__" : { 
        "__base__" : { 
            "secret" : "world"
        }
    }
}
print(son_a.secret)
#hello
print(instance.secret)
#hello
merge(payload, instance)
# k:__class__      v:{'__base__': {'secret': 'world'}}
# getattr(dst,k):  <class '__main__.son_b'>
# k:__base__       v:{'secret': 'world'}
# getattr(dst,k):  <class '__main__.father'>
# k:secret         v:world
print(son_a.secret)
#world
print(instance.secret)
#world

这段代码的关键就是merge函数，简单来说就是将son_()与payload拼接起来，作用与son_b().__class__.__base__.secret = "world"相同，为object添加一个名叫secret且值为world的属性，原本son_a的值是hello，但假设我们访问son_a().__class__.__base__.secret时，访问的便是object中的secret属性的值，就也便可以得到world值。

在这个列子当中存在父类，我们可以用__class__.__base__访问其父类，但在大部分情况下，不会存在父类，这个时候我们就需要依靠全局变量了。

全局变量

与ssti大致使用全局变量时就与ssti差不多相同了，同样是通过__globales__属性来修改想要修改的变量值。
